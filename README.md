##  Project Overview

This project presents an **interactive Coffee Shop Sales Dashboard** built using **Excel, SQL, and Power BI**. The dashboard provides insights into sales performance, customer ordering patterns, product categories, store locations, and time-based trends.

The goal of this project is to demonstrate **end-to-end data analysis**, from raw data handling to visualization and business insights.
